# 电子病历系统补充计划书

## 一、当前实现状态

电子病历服务 (`his-emr`) 已实现基础功能：
- SOAP 结构化病历书写（主观/客观/评估/计划）
- 病历模板管理（通用/儿科/外科/产科模板）
- 三级质控流程（时限/完整性/逻辑质控）
- CDSS 临床决策支持（药物过敏/相互作用/剂量校验）
- 病历列表查询与状态流转

## 二、待补充功能
