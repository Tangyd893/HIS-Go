## SEC-2 接入详情（阶段 8 · 2026-06-10）

| 层 | 位置 | 保护方式 |
|----|------|----------|
| 网关层 | `backend/cmd/gateway/main.go` | `RequireRole("admin")` — `/api/system/*`、`/api/statistics/*` |
| 微服务层 | `backend/cmd/user/main.go` | `RequirePerm("patient:write")` — `POST/PUT/DELETE /api/user/patients` |

权限体系（来自 seed）：
- `admin` → `system:admin`（全权限）+ `statistics:view`
- `doctor` → `patient:read` + `clinic:manage` + `prescription:manage` + `examination:manage` + `emr:manage` + `followup:manage`
- `nurse` → `patient:read` + `inpatient:manage` + `followup:manage`
- `pharmacist` → `pharmacy:dispense` + `prescription:manage`
- `patient` → `patient:read`

验证方式：非 admin 角色 token 访问 `/api/system/*` 应得 403；非 write 权限角色 token 调用 `POST /api/user/patients` 应得 403。
